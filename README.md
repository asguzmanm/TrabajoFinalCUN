# Análisis de la deserción y repitencia escolar en Colombia (2018–2024)

Este repositorio contiene el código desarrollado para el análisis de la deserción y repitencia escolar en Colombia durante el periodo 2018–2024, así como la construcción de un modelo de clasificación orientado a la identificación de municipios con alto riesgo de deserción escolar.

El proyecto se desarrolla como parte de un trabajo académico en análisis de datos y se fundamenta en el uso de información oficial proveniente de fuentes públicas del Estado colombiano.

---

## Objetivo del proyecto

Analizar la evolución y relación entre la repitencia y la deserción escolar en los distintos niveles educativos (primaria, secundaria y media), incorporando el contexto territorial, y desarrollar un modelo predictivo que permita identificar territorios prioritarios para la intervención orientada al fortalecimiento de la permanencia estudiantil.

---

## Metodología

El desarrollo del proyecto sigue la metodología **CRISP-DM (Cross-Industry Standard Process for Data Mining)**, estructurando el trabajo en las siguientes fases:

1. Comprensión del problema educativo  
2. Comprensión y preparación de los datos  
3. Análisis descriptivo y exploratorio  
4. Modelado estadístico y predictivo  
5. Evaluación e interpretación de resultados
