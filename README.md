# Análisis de la deserción y repitencia escolar en Colombia (2018–2024)

Este repositorio contiene el código desarrollado para el análisis de la deserción y repitencia escolar en Colombia durante el periodo 2018–2024, así como la construcción de un modelo de clasificación orientado a la identificación de territorios con alto riesgo de deserción escolar.

El trabajo se desarrolla como parte de un proyecto académico de análisis de datos y utiliza información oficial proveniente de fuentes públicas del Estado colombiano.

---

## Objetivo del proyecto

Analizar la relación entre la repitencia y la deserción escolar en distintos niveles educativos y contextos territoriales, e implementar un modelo predictivo que permita identificar municipios prioritarios para la intervención orientada al fortalecimiento de la permanencia estudiantil.
